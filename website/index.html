PARKPILOT

Autonomous obstacle avoidance, self parking, and cloud telemetry on constrained embedded hardware.

Drive. Avoid. Park. Cloud.

Open Full Proposal

Drive

Direct manual control through IR input with immediate actuator response.

  • Directional steering from remote buttons
  • Speed adjustment from remote keys
  • Live OLED mode and proximity display

Avoid

Continuous obstacle avoidance loop using deterministic threshold logic.

  • Forward scan with front threshold gate
  • Left-right clearance comparison
  • Reverse and rescan when blocked

Park

Parallel parking state machine with cloud-guided and local fallback paths.

  • Right-side wall following and gap detection
  • Guidance request before park maneuver
  • Sensor-based fine adjustment at completion

Cloud

AWS pipeline for telemetry, parking guidance, and collision alerting.

  • Device shadow updates for mode and distances
  • Lambda guidance generation and S3 retrieval
  • SNS alert on collision threshold trigger

PROCESS VISUALIZATION

How ParkPilot executes from input to motion

01

Mode Select

IR command selects Manual, Auto Avoid, or Self Park.

02

Sense

Arduino streams ultrasonic distances and CC3200 samples BMA222.

03

Decide

CC3200 state machine computes steering and speed decisions.

04

Actuate

UART motor command drives PWM steering servo and motor output.

05

Sync Cloud

Shadow update publishes mode and sensor data to AWS IoT.

06

Guide + Alert

Lambda guidance and SNS collision alerts feed back into operation.
