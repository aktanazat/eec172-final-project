Creative Direction

Not a lock. Not a clone. A physical cloud game system.

Radar Arena

Stepper-driven scan head sweeps sectors while ultrasonic range traces threat motion. RGB and buzzer layers create a high-pressure control room feel.

Two Roles

Defender uses joystick for shield placement and remote actions. Attacker injects cloud-driven wave pressure and timed anomalies.

Adaptive Difficulty

Lambda scores reaction latency and sensor anomalies to tune next-round mission intensity in real time.

Replay Data

Every round emits structured telemetry for deterministic replay, adjudication, and leaderboard ranking from S3 artifacts.

Runtime

Eight-stage competitive loop

01

Boot + Sync

Load baseline config and sync shadow state.

02

Calibrate

Baseline ultrasonic, light, tilt, and ambient thermal readings.

03

Fetch Mission

IoT event triggers Lambda mission generation and S3 mission handoff.

04

Threat Sweep

Stepper scan produces sector map and risk gradients.

05

Defend

Joystick + IR actions steer servo shield and countermeasure logic.

06

Score Stream

Telemetry packets publish hit/miss, response latency, and stability metrics.

07

Cloud Judge

Lambda computes winner, penalty model, and next wave seed.

08

Replay + Rank

S3 stores round replay JSON and leaderboard snapshot.

AWS Core

IoT, Lambda, and S3 are in the critical path

IoT Core

  • Shadow desired/reported controls match phases and mission versions
  • MQTT topics carry live telemetry, outcomes, and control events
  • Rules fan events into Lambda pipelines

Lambda

  • mission_generator creates next-wave JSON from performance history
  • score_engine computes weighted score with latency penalties
  • replay_builder composes portable run artifact for S3

S3

  • Mission files versioned by run ID and timestamp
  • Telemetry replay files persisted for grading and reproducibility
  • Leaderboard JSON served for project webpage panel

Embedded Split

CC3200-first architecture with deterministic coprocessing

CC3200 Main Controller

  • Global state machine and competitive game loop
  • AWS IoT TLS client, shadow sync, HTTPS mission fetch
  • SPI display output through 74HC595 pipeline
  • I2C peripheral control and sensor baseline manager

UNO Real-Time Coprocessor

  • Stepper scan timing and SG90 shield actuation
  • Ultrasonic sampling and noise-filtered sector reduction
  • Low-latency actuator commands from UART frames
  • Failsafe local halt if command stream stalls

Sensor + UI Surface

  • Ultrasonic, photoresistor, DHT11, tilt, IR receiver
  • Joystick + buttons for defender controls
  • LCD1602 + 4-digit display + RGB + buzzer for operator feedback
